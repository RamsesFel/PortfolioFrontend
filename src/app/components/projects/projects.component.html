Elden Ring Database

Tech Stack: SQL, C#, Rest API, Web API, Javascript, Typescript, Angular, HTML, CSS, OOP

GitHub: Front-End Back-End SQL

Description: This website is based on the video game Elden Ring and allows players to upload and share their in game characters for other users to copy to try out on their own playthroughs. The intent of this website is to give users new ideas and insperations to replay through the game or help other play the game by suggesting builds that may be easier for new players or builds that are specifically designed to be better aganinst certain enemies. This website would be benefital to anyone who enjoys the game by giving new ideas for players that want to replay the game, giving helpful builds to people struggling against certain enemies, and allowing veteraen players the ability to post their best builds for certain playstyles.

Social Events Website

Tech Stack: SQL, C#, Rest API, Javascript, Angular, HTML, CSS, OOP

GitHub: Front-End Back-End SQL

Description: This website allows users to share events that are happening with other users and allows those users to add those events to their profile page in order to keep track of them. Users can search for different events through a search filter and favorite any events that they find interesting or may want to attend. Should an event get canceled or changed in any way that change will reflect on all users pages so they can remain up to date.
